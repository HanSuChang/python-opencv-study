
## 첫 번째 실습과제 : 그림판 만들기

practice.py 요약

캔버스 설정: 512x512 크기의 흰색 빈 이미지를 생성

마우스 이벤트 처리 (on_mouse 함수):

왼쪽 버튼 클릭 (EVENT_LBUTTONDOWN): 그림을 그리기 시작할 좌표(oldx, oldy)를 저장합니다.

마우스 이동 (EVENT_MOUSEMOVE): 왼쪽 버튼을 누른 상태로 움직이면, 이전 좌표와 현재 좌표를 검은색 선(cv2.line)으로 이어 실시간으로 그립니다.

키보드 단축키 기능:

q 또는 ESC: 프로그램을 종료하고 모든 창을 닫습니다.

c (Clear): 도화지를 다시 흰색으로 싹 지워 초기화합니다.
