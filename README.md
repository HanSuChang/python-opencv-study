
## 첫 번째 실습과제 : 그림판 만들기

OpenCV 실습 01: 마우스 이벤트 활용 그림판 만들기

이 프로젝트는 OpenCV의 마우스 콜백(Mouse Callback) 기능을 활용하여 사용자가 화면 위에 직접 선을 그릴 수 있는 간단한 인터페이스의 그림판을 구현한 실습입니다.

주요 기능 및 로직

마우스 이벤트 처리: cv2.setMouseCallback을 통해 마우스의 클릭, 이동, 떼기 이벤트를 감지합니다.

연속 선 그리기:

LBUTTONDOWN: 클릭한 시점의 좌표를 기록(oldx, oldy)합니다.


MOUSEMOVE + FLAG_LBUTTON: 마우스를 누른 채 움직일 때, 이전 좌표와 현재 좌표를 cv2.line으로 연결하여 끊김 없는 선을 그립니다.



그린 후 현재 좌표를 다시 oldx, oldy에 저장하여 다음 선의 시작점으로 만듭니다.


캔버스 초기화: 특정 키(c)를 누르면 넘파이 슬라이싱(img[:] = 255)을 통해 화면을 즉시 하얗게 지웁니다.


실시간 렌더링: while 루프 내에서 cv2.waitKey(1)을 사용하여 사용자의 키 입력과 마우스 동작을 실시간으로 반영합니다.

사용된 핵심 API 및 제어키

API:

cv2.setMouseCallback(): 마우스 동작 감지 및 핸들러 연결


cv2.line(): 두 점 사이의 선 그리기 (cv2.LINE_AA로 부드러운 안티앨리어싱 적용)


np.ones(): 흰색 배경의 빈 도화지(Canvas) 생성


Controls:

마우스 왼쪽 드래그: 선 그리기

C: 캔버스 초기화 (Clear)

Q / ESC: 프로그램 종료




## 두 번째 실습과제 : 크로마키 만들기

OpenCV 실습 02: 크로마키(Chroma Key) 합성
이 프로젝트는 그린 스크린(Green Screen) 배경에서 촬영된 인물 사진을 추출하여 새로운 배경 이미지(Lena)에 자연스럽게 합성하는 실습입니다.

 주요 기능 및 워크플로우

이미지 전처리: 배경 이미지(lena.jpg)의 크기에 맞춰 인물 이미지(22.jpg)를 리사이즈합니다.


색상 공간 변환 (BGR → HSV): 빛의 영향을 덜 받는 HSV 색상 공간을 사용하여 초록색 배경의 범위를 정확하게 지정합니다.


마스크 생성: cv2.inRange()를 이용해 초록색 배경 부분만 추출하고, 이를 반전시켜 인물만 선택하는 마스크를 만듭니다.


비트 연산 (Bitwise Operations):


bitwise_and를 통해 인물 이미지에서는 사람만 떼어내고, 배경 이미지에서는 사람이 들어갈 자리를 검게 파냅니다.


이미지 합성: 두 이미지를 가중치 합(cv2.addWeighted)으로 결합하여 최종 결과물을 생성합니다.


사용된 핵심 API

cv2.cvtColor(): 색상 공간 변환


cv2.inRange(): 특정 색상 범위의 마스크 생성


cv2.bitwise_and(): 마스크를 이용한 이미지 추출 및 합성 준비


cv2.addWeighted(): 두 이미지의 픽셀 값 결합


실행 결과


1_Extracted Person: 초록 배경이 제거된 인물 이미지


2_Lena with Hole: 인물이 들어갈 자리가 비어있는 배경 이미지


3_Final Result: 완벽하게 합성된 최종 크로마키 이미지



## 세 번째 실습과제 : 문서 스캐너 만들기

OpenCV 실습 03: 원근 변환을 이용한 문서 스캐너
이 프로젝트는 기울어진 각도에서 촬영된 문서(책, 명함 등)의 네 모서리를 마우스로 선택하여, 정면에서 바라본 듯한 반듯한 직사각형 이미지로 보정하는 문서 스캐너 기능을 구현합니다.

화면 맞춤형 리사이징: 고해상도 이미지가 모니터 밖으로 나가지 않도록 가로 길이를 기준으로 비율을 유지하며 cv2.resize를 수행합니다.

마우스 이벤트 좌표 수집:

사용자가 문서의 네 꼭짓점을 클릭하면 해당 좌표를 리스트에 저장합니다.

시각적 피드백을 위해 클릭한 지점에 빨간색 원(cv2.circle)을 그립니다.

원근 변환(Perspective Transform):


입력 좌표(pts1): 사용자가 클릭한 네 지점.


출력 좌표(pts2): 결과창의 직사각형 네 모서리 (600 X 800).


cv2.getPerspectiveTransform을 통해 3x3 변환 행렬 $M$을 계산합니다.


이미지 뒤틀기(Warping): 계산된 행렬 $M$을 cv2.warpPerspective 함수에 적용하여 이미지를 반듯하게 펴줍니다


사용된 핵심 API 및 설정
API:

cv2.getPerspectiveTransform(): 4개의 점 쌍으로부터 원근 변환 행렬 추출

cv2.warpPerspective(): 이미지에 원근 변환 적용

cv2.setMouseCallback(): 마우스 클릭 좌표 수집 핸들러 설정



설정값:결과 크기: 600(가로) X 800(세로) 픽셀

좌측 상단 (Top-Left)

우측 상단 (Top-Right)

좌측 하단 (Bottom-Left)
